# JobLoopAI

JobLoopAI is an AI-powered job search operations agent.

The system connects to Gmail and Google Calendar, automatically tracks recruiting pipelines, identifies deadlines, surfaces follow-ups, and helps job seekers stay organized throughout the interview process.

## Vision

Connect Google once. Never manually track your job search again.

## MVP

- Gmail integration
- Calendar integration
- Recruiting pipeline detection
- Interview tracking
- Assignment tracking
- Follow-up recommendations
- Pipeline health dashboard
